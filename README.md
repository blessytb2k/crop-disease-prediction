 Crop Disease Prediction using Deep Learning

This project focuses on detecting and classifying plant diseases using Deep Learning techniques. It is built using the PlantVillage dataset, which contains labeled images of healthy and diseased plant leaves.

The model is trained to accurately predict the type of disease from leaf images. Beyond prediction, the system also provides detailed information about the detected disease, including its description, possible causes, precautionary measures, and recommended solutions.

This makes the project not only a classification tool but also a practical decision-support system for farmers and agricultural use.

Key Features

* 🌱 Image-based crop disease detection
* 🤖 Deep Learning model for accurate classification
* 📊 Trained on PlantVillage dataset
* 📖 Provides disease description and insights
* 🛡️ Suggests precautions and treatment solutions

Goal

To assist farmers and agricultural communities by enabling early disease detection and providing actionable guidance to reduce crop loss and improve productivity.“Integrated a prediction pipeline with an advisory system that provides actionable insights including symptoms, treatment, and preventive measures.”
